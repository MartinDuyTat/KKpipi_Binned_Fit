Detailed Description

PhaseSpaceParameterisation is a class that contains the information about how phase space is divided into bins PhasespaceParameterisation contains a very coarse and arbitrary binning of phase space A more sophisticated binning can be added by added a new class that inherits from PhaseSpaceParameterisation
